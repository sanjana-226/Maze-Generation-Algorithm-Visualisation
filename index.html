Maze Generation


Recursive Backtracking Maze Generation Algorithm

  1. Start with a grid of cells, all unvisited.
  2. Choose a random starting cell and mark it as visited.
  3. While there are unvisited cells:
    • a. Choose a random neighbor that hasn't been visited yet.
    • b. If all neighbors have been visited, backtrack to the previous cell.
    • c. Mark the chosen cell as visited and add it to the maze path.
  4. When all cells have been visited, the maze is complete.
